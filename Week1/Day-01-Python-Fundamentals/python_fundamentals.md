## What is Python?
Python is an interpreted, high-level, and easy-to-learn programming language.

---

## Interpreted Language
Python executes code line by line.

Example:
print("Hello")
print("Suman")

---

## Print Function
Used to display output.

Example:
print("Hello World")

---

## Input Function
Used to take input from user.

Example:

print("Hello", name)

## Indentation in Python:
Indentation means giving space at the beginning of a line.

In Python, indentation is very important because it defines blocks of code.

Example:

x = 10
y = 20

if x < y:
    print("y is greater than x")
    print("This is inside if block")

print("This is outside if block")

Note:
- Python uses indentation instead of {} (like Java/C++)
- Wrong indentation will give error

## Interpreter vs Compiler:

### Interpreter:
- Executes code line by line
- Stops when error occurs
- Slower but easy to debug

Example:
Python

---

### Compiler:
- Converts whole code at once into machine code
- Shows all errors together
- Faster execution

Example:
Java (uses javac compiler)

---

## Difference Table:

| Feature        | Interpreter (Python) | Compiler (Java) |
|---------------|---------------------|-----------------|
| Execution     | Line by Line        | Whole Program   |
| Speed         | Slower              | Faster          |
| Error Handling| Stops at error      | Shows all errors|
| Output        | Immediate           | After compilation|
